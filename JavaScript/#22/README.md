# 22. Generate Parentheses 產生括號

## Origin

### Question

Given _n_ pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

### Example

```
Input: n = 3
Output: 
[
  "((()))",
  "(()())",
  "(())()",
  "()(())",
  "()()()"
]
```

---

## 翻譯

### 問題

給定 *n* 對的括號, 編寫一個函數來產生所有格式正確的括號的組合.
